hexo备份恢复

发表于 2017-04-11 | 分类于 Hexo
Hexo,你值得拥有~~

备份整个hexo目录

tar czvf hexo.tar.gz /hexo

恢复

Github更换公钥秘钥

重新安装node.js，git,hexo

| 参考：
Hexo-搭建与配置

解压hexo备份

tar xf hexo.tar.gz

删除/hexo目录下几个目录和文件

rm -rf _config.yml source/ themes/

复制解压目录下面的相应文件和目录到/hexo

/bin/cp  -rf hexo/{_config.yml,source/,themes/} /hexo

检查

hexo s

部署

hexo d -g
